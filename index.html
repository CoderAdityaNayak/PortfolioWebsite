Project Printango: HyperLocal Printing Service

Printango Demo Video

Best Project Award — IDT exhibition recognition (from CSE A)

Current Status: Feature-complete; scaling for user testing

Printango Web - For general public

Printango CoPilot - Automation for Shopkeepers


The Problem & Opportunity
Problem: Printing services are scattered — users must visit physical shops or use multiple platforms to find, compare, and order prints
Our Solution: Hyperlocal print shop discovery platform connecting customers with nearby printing vendors
Market: Underserved segment of students, creators, and small businesses needing accessible, affordable printing

Frontend: HTML/CSS/JavaScript
Backend: Supabase (authentication, database, RLS policies)
Maps: Leaflet.js (geolocation and shop discovery)
Documents: PDF.js (document preview and handling)
Automation: Python PyAutoGUI CoPilot
Authentication: Google OAuth via Supabase
